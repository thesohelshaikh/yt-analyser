# yt-analyser

Shows youtube playlist durations in one click

![Banner](playstore/banner.png)

## Features

- Share a URL directly or paste Youtube URL video
- Offline caching
- Dark mode support

## Technologies Used

- Youtube Data API
- Retrofit
- kotlinx.serialization
- Jetpack Compose
- Compose Navigation
- Material 3
- Coil
- Coroutines
- MVVM
- Room Database
